# Trail Time Predict V0.4：个人表现诊断系统开发规划

## 1. 背景

当前系统已经具备：

-   FIT 数据解析
-   GPX 路线分析
-   平路/上坡/下坡能力画像
-   疲劳模型
-   心率模型
-   温度模型
-   Monte Carlo 概率预测

下一阶段目标：

从：

> 比赛时间预测器

升级为：

> 个人表现诊断系统（Performance Analyzer）

核心问题：

不是只回答"这场能跑多久"，而是回答：

"为什么这场跑成这样？"

------------------------------------------------------------------------

# 2. 设计原则

不依赖同路线复盘。

原因：

越野比赛重复路线概率较低。

传统复盘：

同路线 GPX + 比赛 FIT

实际使用频率低。

新的设计：

任何历史 FIT 都可以分析。

输入：

-   历史 FIT
-   个人能力模型
-   可选 GPX
-   可选天气

输出：

-   表现是否符合预期
-   偏差原因
-   下一步训练建议

------------------------------------------------------------------------

# 3. 功能模块

新增：

    Performance Analyzer

    ├── 单次活动表现分析
    ├── 预测偏差分析
    ├── 原因归因

    └── 训练建议

# 4. 单次活动表现分析

输入：

最低：

-   FIT 文件

可选：

-   GPX
-   比赛信息
-   天气数据

输出：

示例：

    实际时间：
    8:12

    模型预测：
    7:45

    差异：
    +27分钟

------------------------------------------------------------------------

# 5. 表现偏差拆解

## 上坡分析

比较：

-   预测爬升时间
-   实际爬升时间
-   不同坡度表现

示例：

    上坡：

    预测：
    3小时10分钟

    实际：
    3小时18分钟

    影响：
    +8分钟

------------------------------------------------------------------------

## 平路分析

分析：

-   平路速度
-   心率成本
-   是否发挥正常

------------------------------------------------------------------------

## 下坡分析

分析：

-   下坡速度
-   技术下降能力
-   坡度对应表现

------------------------------------------------------------------------

# 6. 后程疲劳分析

核心功能：

判断后半程是否超过个人正常衰减。

分析：

-   前后半程速度变化
-   HR 漂移
-   地形能力下降
-   配速变化

示例：

    0-25km：

    符合模型


    25-50km：

    速度下降15%

    HR增加12bpm


    判断：

    后程疲劳高于历史水平

    影响：
    约20分钟

------------------------------------------------------------------------

# 7. 环境影响分析

结合已有：

## 温度模型

输出：

    比赛温度：
    28℃

    最佳温度：
    10-18℃

    预计影响：
    10分钟

## 心率成本

分析：

同样输出下：

实际 HR 是否更高。

------------------------------------------------------------------------

# 8. 模型个人校准

目标：

让模型知道自己的误差。

收集多场活动：

    预测时间
    vs
    实际时间

输出：

例如：

    最近10次：

    平均实际时间比预测慢8%

    模型偏乐观

生成：

个人校准系数。

用于未来预测：

    最终预测 =
    基础预测 × 个人校准系数

------------------------------------------------------------------------

# 10. 训练建议

根据诊断结果生成：

## 后程疲劳问题

建议：

-   增加长距离山地训练
-   提升5小时以上耐力

## 下坡能力不足

建议：

-   增加技术下降训练
-   加强股四头离心力量

# 13. 验收标准

系统能够回答：

## 为什么这场慢？

输出：

    主要原因：

    1. 后程疲劳
    贡献20分钟

    2. 高温
    贡献10分钟

    3. 爬坡发挥
    贡献8分钟

## 模型准不准？

输出：

    最近10次：

    平均误差6%

    模型略偏乐观

## 下一步练什么？

输出：

    当前限制：

    长时间爬升能力

    建议：

    增加4小时山地训练

------------------------------------------------------------------------

# 14. 最终目标

Trail Time Predict 不只是：

比赛预测工具。

而是：

    记录一次活动

    ↓

    理解一次表现

    ↓

    发现能力变化

    ↓

    指导训练

    ↓

    提升下一场比赛成绩

成为：

个人越野数字教练。
